# Complexité

- Prefix, préfixer un élément au début de la collection
- Insert, insérer un élément dans la collection
- Suffix, ajouter un élément à la fin de la collection

- Remove first
- Remove, supprimer un élément dans la collection
- Remove last

- Search by index
- Search by value
